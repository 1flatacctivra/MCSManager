### Minecraft Manager
简单，易用，简约，轻量级的 Minecraft 控制面板



#### 快速上手
我们用最简单上手的方式，来构建整个应用程序，绝大部分的开发者可以轻而易举的进行修改，以及依照协议开发使用。

#### 构建二次开发环境

git clone https://github.com/Suwings/Mc-manager.git

- `Node.js` >= 8.0
- `Java`    >= 7.0 但强烈推荐 Java 8
